365 Days To-Do List Project

This is a simple command-line To-Do List application built in Python.
It allows you to add tasks, mark them as complete, and view all tasks in a list.

Features:
- Add Task: Add a new task to the list.
- Mark as Complete: Mark a task as done.
- View Tasks: See all tasks with their completion status.
- Menu-driven: Easy-to-use numbered menu.
- Infinite Loop: Keep using the program until you choose to quit.

How to Use:
1. Clone the repository.
2. Run the program using: python main.py
3. Follow the menu:
   1- Add task to a list
   2- Mark task as complete
   3- View tasks
   4- Quit
4. Enter the number of the action you want to perform.

Example:
1- Add task to a list
2- Mark task as complete
3- View tasks
4- Quit
Enter your choice: 1
Enter task: Finish homework
Task added to the list successfully!

Notes:
- Tasks are stored in memory and will be lost when the program exits.
- Completion status is shown using ✅ (completed) and ❌ (not completed).
